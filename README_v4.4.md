# 🚀 Nexus IDE v4.4 - The God Update

Welcome to the most powerful version of Nexus IDE yet! **v4.4** introduces a suite of new features designed to make you a 10x developer.

---

## 🔥 Key Features

### 🤖 AI That Works for You

*   **Custom Instructions:** New, fine-tuned instructions for the AI assistant in Chat and Agent modes give you more control over the AI's behavior.
*   **Say /yes to Ship:** Are you tired of confirming every little change the AI wants to make? Just type `/yes` in the chat, and the AI will have full control to build, test, and deploy your code.
*   **Prototyper Mode:** The new Composer/Vibe mode allows you to build entire applications from a single prompt. Just describe what you want to build, and the AI will do the rest.

### 🌐 A True VS Code Experience

*   **VSIX Extension Support:** Install and use your favorite VS Code extensions directly in Nexus IDE. No more compromises.
*   **Beginner Friendly UI:** A new, simplified UI for beginners makes it easier than ever to get started with Nexus IDE. The classic UI is still available in the settings for power users.
*   **Working Templates:** The template system has been completely overhauled. Now you can create and use templates that "actually work."

---

## 💻 How to Use the New Features

### 📝 Custom AI Instructions

The AI assistant now has two new modes:

*   **Chat:** for normal conversations with the AI.
*   **Agent:** for "review-based developments." The AI will act as a senior developer, reviewing your code and making suggestions. If you're happy with the suggestions, you can say `/yes` to have the AI implement them for you.

### ⚡ Say /yes to Ship

When the AI suggests a change, you can now type `/yes` in the chat to automatically approve and apply the change. No more clicking buttons!

### 🏗️ Prototyper Mode

To use the new Prototyper mode, simply select "Composer/Vibe" from the AI mode dropdown. Then, describe the application you want to build in the chat. The AI will ask you a few questions to clarify the requirements, and then it will generate the entire application for you.

### 🧩 VSIX Extension Support

To install a VS Code extension, go to the "Extensions" view in the sidebar and search for the extension you want to install. You can also manually install extensions from a `.vsix` file.

### ✨ Beginner Friendly UI

The new beginner-friendly UI is enabled by default. To switch back to the classic UI, go to **Settings > UI > Use Classic UI**.

### 📁 Templates

The new template system is available in the "Explorer" view. To create a new template, simply right-click on a file or folder and select "Save as Template." To use a template, right-click in the "Explorer" view and select "New from Template."
