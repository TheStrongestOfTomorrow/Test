# 🚀 Nexus IDE v4.3 - The Intelligence & Content Update

Welcome to the biggest feature drop in Nexus history! **v4.3** brings AI voice control, advanced project visualization, and productivity tools to supercharge your workflow.

---

## 🎙️ Nexus Voice (New!)
Control your IDE with your voice. Click the microphone icon in the Title Bar to start.
*   **"Run Code"**: Executes the current file.
*   **"Open Settings"**: Opens the settings panel.
*   **"Toggle Terminal"**: Shows/hides the terminal.
*   **"Clear Workspace"**: Wipes all files (safely prompts first).
*   **"Analyze"**: Triggers AI architecture analysis.

## 🧘 Zen Mode
Focus on what matters—your code.
*   **Toggle**: Click the Eye icon in the Title Bar.
*   **Effect**: Hides the Sidebar, Terminal, Activity Bar, and extra UI elements.

## 🎨 Theme Studio
Create your own vibe.
*   **Location**: Sidebar -> Palette Icon.
*   **Features**: Customize background, accent, and text colors in real-time.

## 📦 Dependency Visualizer
See what powers your project.
*   **Location**: Sidebar -> Package Icon.
*   **Features**: Visual breakdown of `dependencies` vs `devDependencies` from `package.json`.

## 📝 Todo Scanner
Never lose track of a task.
*   **Location**: Sidebar -> CheckSquare Icon.
*   **Features**: Automatically finds and lists all `// TODO`, `// FIXME`, and `// HACK` comments. Click to jump to line.

## ⏱️ Pomodoro Timer
Stay in the zone.
*   **Location**: Status Bar (Bottom Center).
*   **Features**: 25-minute focus timer with Play/Pause/Reset controls.

## 💾 Snippet Manager
Code faster.
*   **Location**: Sidebar -> Scissors Icon.
*   **Features**: Save frequently used code blocks and copy them with one click.

## 📊 Project Insights
Know your codebase.
*   **Location**: Sidebar -> BarChart Icon.
*   **Features**: Visual stats on total lines of code, file types, and language breakdown.

## 🚀 Live App Preview 2.0
Not just a file list anymore.
*   **Smart Serving**: Automatically finds and serves `index.html`.
*   **Injection**: Inlines local CSS/JS to make apps run without a backend.
*   **Python Support**: Runs `.py` files using Pyodide (WASM) directly in the browser.

## 🔢 Line Counter
*   **Location**: Status Bar (Bottom Right).
*   **Features**: Real-time counter of total lines of code (LOC) in your workspace.
